# Devears

- Devears는 `Developer + Bears`를 의미합니다.
- `Bears`는 단국대의 상징인 `곰`을 의미합니다.
- 쉽게 말해서 단국대 개발자들이라고 보면 될 것 같습니다.

프로젝트에 관련된 전반적인 내용은 [docs](./docs)에 정리해놓았습니다.

## 사용방법

### 개발자에게 업무 요청
- Projects 탭의 XXX에게 요청하기에서 카드를 작성합니다. 카드를 작성하고 이슈를 생성합니다.
