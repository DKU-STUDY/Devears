### 업무 요청 방법

### PR 작성법 
1. 기능 개발을 시작하기 전, Projects 에 알맞은 프로젝트를 고르고, 기능 카드를 생성합니다. 
2. 기능 카드에 대한 이슈를 생성합니다.
3. Pull Request 작성 후, 기능 카드에 대한 이슈에 PR 을 링크 합니다.
